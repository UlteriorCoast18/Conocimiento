# Knowledge

This repository aims to compile different courses and information about Physics, Mathematics, and Programming, specifically in Mathematics, which is the author's field of study. Here, you will find freely accessible information for people interested in any of the aforementioned topics.

The following is the file organization of all the folders inside the repository, so that anyone can access and focus on the topics they are interested in. All the folders contain a `.pdf` file, or both a `.pdf` and a `.tex` file (even my `.tex` template is available for its use). The reason behind this is that the author believes everyone should have access to information and, furthermore, should be able to modify it as they wish.

Some of the topics in the Mathematics folder are the following:

-   **Mathematical Analysis** (real analysis, some topics about complex analysis, Lebesgue measure, and functional analysis).
-   **Abstract Algebra** (groups, rings, modules, categories, and topological groups).
-   **Differential Geometry** (differential geometry of curves, surfaces and manifolds, differential forms, and integration of differential forms).
-   **Point-Set Topology**.
-   **Algebraic Topology** (homotopy groups, homology groups, and results about free groups).
-   **Logic** (first and second order logic, models, incompleteness and completeness theorems).
-   **Set Theory** (ordinals, cardinals).

And so on.

Also, in the programming field (which is still incomplete but in progress), we have the following programming languages:

-   Java
-   Python
-   C

Not many projects have been developed in these languages yet, but we are planning to add more of them.

**Note:** _The author of this repository studies at the Escuela Superior de Física y Matemáticas (ESFM), which explains the "ESFM Courses" folder containing introductory higher-level mathematics courses._

The `ESFM Courses` folder is organized by semester, with each semester's folder containing the courses taken that term. Each course folder has three subfolders:

- **Exams**: Collection of exams from the course, with solutions when available.
- **Problem Sets**: The main purpose of this repository. Contains exercise lists and their solutions (when successfully solved). These problems can significantly aid mathematical training.
- **Notes**: Course notes numbered sequentially for easy follow-up. Each note includes its topic for quick reference.

**Note:** _If you solve any unsolved problems from the problem sets, please email your solution to `cristo.daniel.alvarado@gmail.com`. Accepted contributions will be credited upon request._
